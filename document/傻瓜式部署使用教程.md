# INTALINK 傻瓜式部署使用教程

## 一、Docker 安装

1. 将下载的压缩包上传到 Linux 服务器上（文件参考 `docker-27.2.1.tgz`）。

2. 解压压缩包：
   ```bash
   tar -xvf docker-27.2.1.tgz
将解压出来的 Docker 文件内容移动到 /usr/bin/ 目录下（可使用移动命令或者复制命令）：

```bash
复制代码
mv docker/* /usr/bin/
或

```bash
复制代码
cp docker/* /usr/bin/
将 Docker 注册为 service 服务：需要根据自己的配置来编写 docker.service，参考附件 1：

bash
复制代码
vi /etc/systemd/system/docker.service
启动 Docker：

bash
复制代码
chmod +x /etc/systemd/system/docker.service  # 设置权限
systemctl daemon-reload
systemctl --now start docker  # 启动 Docker
使用以下命令检查 Docker 是否安装成功：

bash
复制代码
docker version
二、Docker-compose 安装
将 docker-compose 文件放在 /usr/local/bin/ 目录下。

使用以下命令检查 docker-compose 是否安装成功：

bash
复制代码
docker-compose --version
三、系统搭建
从 Docker Hub 上拉取镜像：
