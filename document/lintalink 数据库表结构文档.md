## 1. 链路生成方法表

| 列名                   | 数据类型    | 是否为空 | 主键 | 自增 | 默认值 | 备注     |
|-----------------------|-----------|---------|------|------|-------|----------|
| generating_methods_id | int       | 否      | 是   | 是   |       | 主键     |
| generating_methods_name | varchar(255) | 是    | 否   | 否   |       | 方法名称 |
| implementation_method | varchar(1000) | 是   | 否   | 否   |       | 实现方式 |
| implementation_team   | varchar(100) | 是    | 否   | 否   |       | 实现队伍 |

## 2. 数据申请日志

| 列名             | 数据类型   | 是否为空 | 主键 | 自增 | 默认值 | 备注     |
|-----------------|-----------|---------|------|------|-------|----------|
